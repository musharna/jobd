- **Test/CI hygiene sweep.** The CI coverage gate now measures **branch** coverage (line coverage alone lets an untested `else` arm count as covered because its `if` line ran) with the floor recalibrated to 83 (85 measured), and a deploy-lint keeps both the floor and `--cov-branch` from silently disappearing. The one standard broker-fixture body that had been pasted into 24 test files is a single set of `app`/`client`/`client_logs` fixtures in `tests/conftest.py`. A drain test that gave a cold `python3` workload a fixed 0.5s to install its SIGTERM-ignoring handler now waits for the workload's READY byte — firing early made the SIGKILL-cap path under test pass vacuously. The README platform table now says what CI actually verifies (Linux) instead of implying macOS/Windows are tested.
